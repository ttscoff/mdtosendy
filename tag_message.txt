### 1.0.0

2025-12-26 04:23

#### NEW

- YAML frontmatter in Markdown files can now override configuration settings from config.yml, allowing per-email customization without modifying base configuration
- Frontmatter supports flat keys (e.g., header_image_url) that automatically map to nested config structure (e.g., template.header_image_url)
- Frontmatter also supports nested structure (e.g., template: { header_image_url: ... }) for explicit configuration overrides
- Frontmatter values take precedence over config.yml values, enabling individual email customization while maintaining base settings
